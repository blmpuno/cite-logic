## CITE Logic: A Java Console Project
In partial fulfillment of the subject **IT105 - Discrete Mathematics**.

## Description
CITE Logic is a Java-based CLI application that demonstrates core concepts of **Propositional Logic** through truth tables, logical evaluation, and an interactive quiz game.

## Features
This system performs the following discrete mathematics operations:

### Truth Table Generator
- Automatically generates complete truth tables for logical expressions.

### Manual Logical Evaluation
- Allows users to assign truth values (T/F) to variables and compute results.

### Logical Expression Builder
- Supports common logical operators:
  NOT  
  AND  
  OR  
  XOR  
  IMPLIES (→)  
  BICONDITIONAL (↔)

### Logic Quiz Game (3 Lives System)
- Answer up to 5 logic-based questions  
- Start with 3 lives  
- Lose a life for every incorrect answer  
- Option to continue or exit after each question  



## System Features

- **Dynamic Input Handling**  
  Users can freely input variables and select logical operators.

- **Automated Evaluation Engine**  
  Expressions are computed using a built-in logical parser and evaluator.

- **Interactive CLI Interface**  
  Simple and user-friendly console navigation.

- **Educational Design**  
  Designed to reinforce understanding of logical operations step-by-step.
